# -Soje-dong-MBTI-
소제동 성향 테스트 (Soje-dong MBTI)
이 프로젝트는 무엇인가요?
사용자의 여행 스타일을 분석하여 소제동의 최적화된 장소를 추천해 주는 심리 성향 테스트 도구입니다.

왜 만들었나요?
소제동을 방문하는 여행객들이 자신의 취향에 맞는 공간을 더 쉽게 찾고, 매번 고민하는 시간을 줄여 더 즐거운 여행 경험을 제공하기 위해 만들었습니다.

어떤 기능이 있나요?
성향 분석: 10가지 문항을 통해 사용자의 여행 스타일을 활동성 점수로 계산합니다.

맞춤 추천: 점수에 따라 5가지 유형(에너자이저, 탐험가, 감성 수집가, 문화 산책가, 골목 사색가)으로 결과를 분류하여 장소를 추천합니다.

결과 확인: 테스트 즉시 자신의 유형과 관련 키워드를 확인할 수 있습니다.
